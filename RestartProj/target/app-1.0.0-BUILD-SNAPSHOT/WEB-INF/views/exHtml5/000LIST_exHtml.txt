1. 웹역사 및 HTML5의 등장
-브라우저의 종류
오페라 : 유럽. 빠른속도, 작은 용량
사파리 : 애플, 빠른속도, 모바일에서 많이 사용
크롬 : 구글, 다른 브라우저와 호환이 떨어짐. 가장 최근에 나옴
파이어폭스 : ActiveX를 우회하여 적용가능함. 
익스플로러 : 윈도우 설치 시 끼워서 같이 됨. ActiveX를 사용하여 플러시 기능. 그러나 너무 무거움
